### TrainingCenter_pkid column

Show Records from TrainingCenter for each tab. Tab Label=TrainingCenter.Name, Tab value= TrainingCenter.pkid 
When a tab is clicked (activated), filter TrainingCenter_pkid column

### ScheduleOn column

One-week, monday to Sunday
Filter ScheduleOn column for the selected week

### Slot column

Slot 1, 2, 3


### Promotion_pkid column

Look up  Promotion2.PromoCode, based on Promotion_pkid

### Topic column

Show Topic column value

###  Description column

Show Description column value
