# CMS

Glossary for the course-training CMS. Terms specific to this domain; no
implementation details.

## Language

**Course Detail**:
The public, student-facing composite view of a single course — the course's own
fields plus its FAQs, related links, job categories, and certifications. This is
what a prospective student browses, not what an editor edits.
_Avoid_: course page, course view (ambiguous with the admin edit form)

**Course PDF**:
A downloadable PDF document of a Course Detail, produced by the backend and
retrievable by any front-end caller. Intended for a student to submit to an
employer for training approval or reimbursement.
_Avoid_: course export, course printout
